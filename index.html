Travel Destination API Documentation

Overview

This API returns a list of travel destination options based on the selected continent.

Base URL

https://midterm26.onrender.com/api/destination_update.php

Request Method

GET

Parameters

Parameter Type Description Example
continent String Continent code AS / EU

Supported Continent Codes

Code Continent
AS Asia
EU Europe

Example Response

Request

GET /api/destination_update.php?continent=AS

Response

<option value='Tokyo, Japan'>1. Tokyo, Japan</option>
<option value='Seoul, South Korea'>2. Seoul, South Korea</option>
<option value='Bali, Indonesia'>3. Bali, Indonesia</option>
<option value='Bangkok, Thailand'>4. Bangkok, Thailand</option>
<option value='Kyoto, Japan'>5. Kyoto, Japan</option>
<option value='Hanoi, Vietnam'>6. Hanoi, Vietnam</option>

Error Responses

Invalid Continent

<option value=''>No destinations found</option>

Missing Parameter

<option value=''>Missing continent parameter</option>

Content Type

text/html
